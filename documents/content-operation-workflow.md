# 内容运营主流程

Sanmoo Blog 的内容运营只服务于个人原创技术内容的持续沉淀，不引入投稿、社区或付费运营链路。

1. 在后台创建文章草稿并完成 Markdown 写作。
2. 为文章选择分类与标签；需要形成系列知识时创建或编排到专题。
3. 通过文章状态和发布时间完成立即或定时发布。
4. 发布内容同时进入 Web 的搜索、归档、SEO/RSS 和小程序的轻阅读列表。
5. 在内容看板查看阅读、分类、标签与专题的聚合表现，用于选题和内容复用。

## 约束

- 相关内容与推荐仅使用文章分类、标签和专题关系。
- 小程序只保留阅读、收藏、历史与订阅；不恢复画像生成或精细化运营入口。
- 所有新内容能力必须能回到上述链路中的一个环节，不能扩展为用户社区或交易能力。
